## ğŸ”§ Technical Details

### Data Structure

Templates stored in `/public/data/templates/`:
- `product-showcase.json` - 9 products
- `product-ordering.json` - 10 products
- `starter.json` - 6 services + testimonials

### Clear Function Logic

```javascript
function clearDemoData() {
  const siteData = window.currentSiteData || {};
  
  // Clear arrays
  if (siteData.products) siteData.products = [];
  if (siteData.services?.items) siteData.services.items = [];
  if (siteData.testimonials?.items) siteData.testimonials.items = [];
  
  // Reset text
  if (siteData.brand) siteData.brand.name = 'Your Business Name';
  if (siteData.hero) {
    siteData.hero.title = 'Your Headline Here';
    siteData.hero.subtitle = 'Your subtitle or tagline';
  }
  
  // Update editor
  window.currentSiteData = siteData;
  if (window.refreshEditor) window.refreshEditor(siteData);
  
  // Save cleared state
  GuestEditor.saveGuestProgress();
}
```

### Auto-Save Integration

- Demo data included in saves
- Cleared state also saved
- Progress persists across sessions
- No conflicts with user data
