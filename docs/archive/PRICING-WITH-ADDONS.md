## üí≥ **Billing Details**

### **How Charges Work:**

**Initial Setup:**
- Subscribe to Starter: $10/month
- Payment: $10 charged monthly

**Add First Add-On:**
- Add 2nd site: +$5/month
- New total: $15/month
- **Pro-rated:** $5 √ó (days left / 30) charged immediately
- Next month: Full $15/month

**Add Second Add-On:**
- Add 3rd site: +$5/month
- New total: $20/month
- Pro-rated: $5 √ó (days left / 30) charged immediately
- Next month: Full $20/month

**Remove Add-On:**
- Remove 1 site: -$5/month
- New total: $15/month
- Credit applied to next invoice

### **Stripe Implementation:**
Each add-on = separate line item on subscription:
```json
{
  "subscription": {
    "items": [
      { "price": "price_starter", "quantity": 1 },      // $10
      { "price": "price_starter_addon", "quantity": 2 }  // $5 √ó 2
    ]
  }
}
```
