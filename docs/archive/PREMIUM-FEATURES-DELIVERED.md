### **To Use Live Chat:**
1. Get API key from provider (Intercom, Drift, etc.)
2. Update template JSON:
   ```json
   "chat": {
     "enabled": true,
     "provider": "tidio",
     "appId": "YOUR_KEY_HERE"
   }
   ```
3. Save and reload - chat widget appears!

### **To Use Enhanced Profiles:**

```json
}
```
