## üöÄ BEST PRACTICES

### For QR Codes on Print:

1. **Size:** Minimum 1" √ó 1" for scanning
2. **Contrast:** High contrast (black on white best)
3. **Placement:** Lower right corner typically
4. **Testing:** Always test scan before mass printing
5. **Call to Action:** Add "Scan to visit" text

### For Print Quality:

1. **Resolution:** Use high-quality PNG export
2. **Colors:** CMYK for professional printing
3. **Bleed:** Add 0.125" bleed if required
4. **Proof:** Always get a proof before bulk order
5. **Paper:** Glossy for photos, matte for professional
