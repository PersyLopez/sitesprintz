## If Something Doesn't Look Right

### 1. Clear Browser Cache
**Chrome/Edge:** `Cmd+Shift+R` (Mac) or `Ctrl+Shift+F5` (Windows)
**Firefox:** `Cmd+Shift+R` (Mac) or `Ctrl+F5` (Windows)
**Safari:** `Cmd+Option+E`, then `Cmd+R`

### 2. Check Server is Running
Server should be running on `http://localhost:3000`

If not, start it:
```bash
cd /Users/admin/active-directory-website
node server.js
```

### 3. Verify File Changes
All template JSON files have been updated. If you don't see changes:
- Hard refresh (step 1)
- Check browser console for errors (F12)
- Verify you're looking at the right template
